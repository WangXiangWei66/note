安装第三方模块PyInstaller模块
执行打包操作
 参数-F表示只生成一个扩展名为.exe的可执行文件)

4.文件读取的seek() 


  1. with open('a.txt','w',encoding='utf-8')as file:
  2.     file.write('中国加油！')
  4. file =open('a.txt','r',encoding='utf-8')
  5. file.seek(3)
  6. print(file.readline())
报错 

  1. with open('a.txt','w',encoding='gbk')as file:
  2.     file.write('中国加油！')
  4. file =open('a.txt','r',encoding='gbk')
  5. file.seek(3)
  6. print(file.readline())

解决方案： 

  1. with open('a.txt','w',encoding='gbk')as file:
  2.     file.write('中国加油！')
  4. file =open('a.txt','r',encoding='gbk')
  5. file.seek(2)
  6. print(file.readline())
